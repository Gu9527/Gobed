# 无基不行 (No Bed No Go)

一个为 Minecraft 1.21.x 添加自定义超平坦维度的 NeoForge 模组。

## 功能

### 维度系统
- 自定义超平坦维度，支持动态方块配置
- 维度传送方块，搭建 3x3 结构即可传送
- 传送至维度后自动在落点放置传送方块，右键即可返回主世界（位置精确保存）
- 可配置区块单位（默认 2 = 32×32 格），每个区域有独立的边框和内侧方块
- 维度地表高度可配置（默认 64），底层为基岩
- 固定正午时间（仅本维度），不影响主世界/下界/末地
- 禁止自然刷怪和结构生成
- 多人服务器下，传送会自动寻找安全落点（避免玩家互相踩头/卡进方块）

### 传送方块
- 搭建 3x3 结构右键传送，结构示意图在 tooltip 中直观显示
- 传送时根据结构中的方块动态更新维度边框/内侧方块配置
- 边框/内侧方块黑名单，禁止使用基岩等特定方块
- 方块自带传送门粒子效果和附魔粒子飘动

### AE 大礼包
- 使用后获得装满物品的便携式 16k 存储元件（AE2 API 实现）
- 内含 AE2 核心物品及扩展模组物品（自动检测已安装模组）
- 彩虹流动渐变色 tooltip 文字
- 高分辨率 256×256 材质的 3D 盒体模型
- 未安装 AE2 时自动回退到直接给物品

### 配置界面
- 支持游戏内实时修改（Mod 列表 → Go Bed → 配置按钮）
- 光照等级、出生高度、区块单位、维度高度
- 边框/内侧方块黑名单（逗号分隔的方块 id）
- 一键启用/禁用传送功能
- 批量校验，任一字段非法时整张配置不会保存

## 安装

1. 确保已安装 NeoForge 21.1.233 或更高版本
2. 将编译好的 mod 文件放入 `mods` 文件夹
3. 可选依赖（部分物品需要对应模组）：
   - AE2 (appliedenergistics2)
   - ExtendedAE / ExtendedAE Plus
   - AE2 WirelessLib / AE2LT
   - Advanced AE / MinforMax / OmniTools
4. 启动游戏

## 使用方法

### 传送方块

1. 合成或获取传送方块（合成配方：圆石围一圈，中间加一个木板）
2. 搭建 3x3 结构：
   ```
   X O X
   O P O
   X O X
   ```
   - P = 传送方块（本方块）
   - O = 内侧方块（任意）
   - X = 边框方块（任意）
3. 右键传送方块传送到自定义维度
4. 在维度内右键传送方块立即返回主世界

### AE 大礼包

1. 获取 AE 大礼包（创造模式物品栏或指令）
2. 右键使用，获得装满物品的便携式 16k 存储元件
3. 元件已充满电，可直接在 AE2 网络中使用

### 配置

在游戏中打开 Mod 列表，选择"Go Bed"，点击配置按钮可修改。

## 编译

```bash
# 使用本地 Gradle 构建
.\gradlew.bat build
# 编译后的 mod 文件位于 build/libs/ 目录
```

## 技术细节

- 维度使用自定义 `ChunkGenerator` 生成超平坦地形
- 底层为基岩，上方由边框方块和内侧方块填充
- 维度方块配置随存档持久化（仅影响新生成区块）
- 便携元件通过 AE2 `AEComponents.STORAGE_CELL_INV` API 填充
- 纹理动画通过 `.png.mcmeta` 配置文件实现多帧循环

## 注意事项

- 需要 NeoForge 21.1.233+
- 需要 Minecraft 1.21.x
- 维度方块配置重启后重置为默认值
