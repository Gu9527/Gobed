# 无基不行 (No Bed No Go)

一个为 Minecraft 1.21.x 添加自定义超平坦维度的 NeoForge 模组。

## 功能

- 自定义超平坦维度，支持动态方块配置
- 维度传送方块，搭建 3x3 结构即可传送
- 2x2 区块单位，每个区域有独立的边框和内侧方块
- 固定正午时间，天气与主世界一致
- 禁止自然刷怪和结构生成
- 支持游戏内配置修改

## 安装

1. 确保已安装 NeoForge 21.1.233 或更高版本
2. 将编译好的 mod 文件放入 mods 文件夹
3. 启动游戏

## 使用方法

### 传送方块

1. 合成或获取传送方块（合成配方：圆石围一圈，中间加一个木板）
2. 搭建 3x3 结构：
   ```
   X O X
   O P O
   X O X
   ```
   - P = 传送方块（本方块）
   - O = 内侧方块（任意）
   - X = 边框方块（任意）
3. 右键传送方块即可传送到自定义维度

### 配置

在游戏中打开 Mod 列表，选择"无基不行"，点击配置按钮可修改：
- 传送方块光照等级 (0-15)
- 传送后出生高度 (-64 到 320)
- 区块单位大小 (1-16)
- 维度地表高度 (1-320)
- 是否启用传送功能

## 编译

1. 确保已安装 JDK 21
2. 运行 `./gradlew build`
3. 编译后的 mod 文件位于 `build/libs/` 目录

## 自定义维度

- 维度使用自定义 ChunkGenerator 生成超平坦地形
- 底层为基岩，上方由边框方块和内侧方块填充
- 维度高度可配置，默认 64 格
- 维度高度范围为 -64 到 320

## 注意事项

- 需要 NeoForge 21.1.233+
- 需要 Minecraft 1.21.x
- 请确保纹理文件已正确放置在 textures 目录中
